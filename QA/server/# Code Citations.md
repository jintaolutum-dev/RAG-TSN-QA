```
最简单的方案是做一个**独立的 HTML 聊天页面**，放在 Flask 的 `static/` 目录下，后端启动后直接访问 `http://localhost:5000/static/chat.html`，自动后台登录、自动选知识库，打开就能聊。

---

把下面这整段代码保存为 [QA/server/static/chat.html](QA/server/static/chat.html)：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>企业知识库问答</title>
  <style>
    * { box-sizing: border-box; margin: 0; padding: 0; font-family: system-ui, sans-serif; }
    body { display: flex; height: 100vh; background: #f5f7fa; }

    /* 左侧知识库列表 */
    #sidebar {
      width: 220px; background: #fff; border-right: 1px solid #e4e7ed;
      display: flex; flex-direction: column
```
